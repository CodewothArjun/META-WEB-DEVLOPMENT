Local Storage

Local Storage is a web storage object which are not sent to server with each request. This data services a full page refresh and even a full browser restart.

Some methods provided by local storage are:

  • setItem(key,value) : Store key/value pair
  • getItem(key) : Get the value by key
  • removeItem(key) : Remove the key with its value
  • clear() : Clear everything
  • key(index) : Get a key on a given position
  • length : The number of storage items
